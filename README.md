# 🖥️ CMS-WordPress-Labs — Лабораторные работы по Content Management Systems

![WordPress](https://images.ctfassets.net/lzny33ho1g45/35AyaIhLYULgCzUA3g8AyY/62b277a5b9ce915d8a937eb23fead7f9/WordPress.jpg?fm=jpg&q=31&fit=thumb&w=1520&h=760)

📌 **Описание:**  
Этот репозиторий содержит лабораторные работы по курсу **Content Management Systems (CMS)**.  
В рамках курса изучаются основы систем управления контентом, установка и настройка **WordPress**, работа с темами, плагинами, контентом и базовыми возможностями администрирования сайта.

---

## 🚀 **Структура репозитория**

- 🔹 [Лабораторная работа №2: Введение в WordPress](lab2-wordpress-installation/) — **`lab2-wordpress-installation/`**

- 🔹 [Лабораторная работа №3: Разработка простой темы WordPress](lab3-simple-theme/) — **`lab3-simple-theme/`**

- 🔹 [Лабораторная работа №4: Разработка плагина для WordPress](lab4-wordpress-plugin/) — **`lab4-wordpress-plugin/`**

- 🔹 [Лабораторная работа №5: Безопасность WordPress](lab5-wordpress-security/) — **lab5-wordpress-security/**

---

## 🛠 **Как начать работу?**

💻 **1. Клонировать репозиторий**

```sh
git clone https://github.com/NikitaBytes/wordpress-cms-course.git
cd wordpress-cms-course
```

**2. Перейти в папку нужной лабораторной**

```sh
cd lab2-wordpress-installation
```

**3. Открыть отчёт, материалы и скриншоты**

- README.md или report.md — описание выполненной работы
- screenshots/ — скриншоты ключевых этапов
- дополнительные файлы — по необходимости
